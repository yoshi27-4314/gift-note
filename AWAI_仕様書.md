## 1. プロダクト概要

### コンセプト
> 「Google広告は自分の欲を刺激する。AWAIは大切な人を喜ばせるヒントをくれる。」

AWAIは、大切な人の情報を記録・管理し、AIがギフトや体験を提案する**Personal CRMアプリ**。贈り物の履歴、相手の好み・記念日・サイズ情報などを一元管理し、最適なギフト提案をAIコンシェルジュが行う。

### ターゲットユーザー
- 贈り物を大切にする個人（家族・友人・恋人への贈答）
- 法人の贈答管理（お中元・お歳暮・取引先管理）
- 接客業（常連客の好み管理）
- BtoB営業（法人顧客管理）

### ビジネスモデル（検討中）
- **短期戦略:** 無料配布でユーザー数を最速で伸ばす。データ回収に全振り
- **収益候補:** ターゲティング広告（確定データ活用）、名店提携送客、ギフトコンシェルジュ枠、プレミアムプラン
- **バイアウト候補:** リクルート、LINE、メルカリ、Sansan、ギフトモール
- **業務提携候補:** ピアリー（羽島、売上55億、社長が知り合い）→ AWAIのパーソナルデータ × ピアリーのギフトEC = 提案エンジン

### 競合優位性
| 項目 | AWAI | 一般的なギフトアプリ |
|------|------|---------------------|
| 人物プロファイル深度 | サイズ/嗜好/推し/食の好み/家族構成/個性 | 名前・誕生日程度 |
| AI提案の精度 | 苦手除外・過去重複回避・個性考慮 | 汎用レコメンド |
| データの所有権 | ユーザー端末内（プライバシー完全保護） | クラウド依存 |
| 法人対応 | お中元/お歳暮/予算管理/名刺写真/所属メンバー管理 | 個人向けのみ |
| 広告モデルの可能性 | 確定データ（購買意図+相手プロフィール） | 推定データ |
| ターゲティング精度 | タグ×年齢×性別×好み＝Google以上の精度 | 推定ベース |

### 3.4 大切な人（個人/法人）

**最も詳細なデータ管理。個人と法人で表示フィールドが自動切替。**

#### 種別の決定方法
- 「すべて」「個人」タブで＋ボタン → 個人として登録
- 「法人」タブで＋ボタン → 法人として登録
- モーダル内での切り替えは不要（直感的）

#### 個人カードのフィールド
| カテゴリ | フィールド |
|---------|-----------|
| 基本 | 呼び名（必須）、本名、関係、性別、顔写真（カメラ/ファイル）、所属法人 |
| 📅記念日 | 名前・日付（年月日/月日/月のみ）・繰り返し（毎年/毎月/単発）・リマインダー（複数日数設定） |
| 📏サイズ | 服トップス / ボトムス / 靴 / 指輪 |
| 🚬嗜好品 | タバコ（銘柄）/ お酒（種類） |
| 💖好きなもの | 趣味・興味（ピックアップ提案に使用） |
| 🎨ブランド | 好みのブランド・色 |
| 🌟推し活 | アーティスト・キャラ・チーム |
| 🍽食の好み | 好きなもの / 苦手なもの |
| 👨‍👩‍👧家族 | 名前＋メモ（複数登録） |
| ✨個性 | 性格特性メモ |
| 🎐🎄季節贈答 | お中元/お歳暮（毎年手配+予算） |
| 📝メモ | 自由記述 |

#### 法人カードのフィールド（個人より簡素化）
| フィールド | 説明 |
|-----------|------|
| 呼び名（必須） | 自分がどう呼んでいるか |
| 正式名称 | 会社名（任意） |
| 📷写真 | 名刺など（カメラ/ファイル） |
| 業種 | 業種分類 |
| 住所 | 送付先 |
| 📅記念日 | デフォルト：創立記念日（日付未設定なら非表示） |
| 🎐🎄季節贈答 | お中元/お歳暮（予算付き） |
| 👥所属メンバー | 自動表示 + ＋追加 / ✕解除ボタン |
| 📝メモ | 自由記述 |

#### 個人⇔法人の相互リンク
- **個人カード** → 所属法人名タップ → 法人カードに移動
- **法人カード** → 所属メンバー一覧表示 → 名前タップで個人カードに移動
- **法人カードから直接追加** → 「＋ メンバーを追加」で個人一覧から選択
- **解除** → メンバー横の✕ボタンで所属解除（確認ダイアログ付き）

#### アイコンの性別色分け
- 女性：ピンクのSVGシルエット
- 男性：青のSVGシルエット
- 未設定：グレーのSVGシルエット
- 法人：🏢

#### カード表示機能
- リスト表示（LINEスタイル）＋ タップで詳細展開
- 「← 一覧に戻る」ボタン
- 「📋 この人との記録」→ あげた/もらった/アイテム/行きたいから自動集約（最大10件）
- 各履歴アイテムをタップ → 該当タブ・カードへジャンプ
- 記念日セクション内「📅 一覧へ」→ 記念日カードセクションにスクロール

### 3.9 セキュリティ

|------|------|

## 7. 機能面のアイデア・アドバイス

### 即効性の高いアイデア
1. **「今日何する？」ホーム画面** — 起動時に今日の記念日・リマインド・おすすめアクションをまとめ表示。今はタブを開かないと何も見えない
2. **データエクスポート/インポート** — JSONファイルで書き出し・読み込み。LocalStorage消失リスクへの保険。バックエンド導入前の必須機能
3. **お返し提案** — もらった記録に「お返しする」ボタン → AIが金額・相手の好みから提案
4. **贈答カレンダー表示** — 記念日・贈答予定をカレンダービューで一覧表示
5. **予算管理ダッシュボード** — 月別・年別の贈答支出グラフ

### 中期的なアイデア
6. **ウィッシュリスト共有** — 自分のほしいものリストをURLで共有。相手がAWAIを入れなくても見れる
7. **贈り物テンプレート** — 「母の日」「父の日」「還暦祝い」等のシーン別テンプレート
8. **相性診断** — 2人のプロフィールからAIが「共通の趣味」「喜ばれるペアギフト」を提案

## 8. デザイン面のアドバイス

1. **カード展開時のアニメーション** — 現在は即時展開。スライドダウンアニメーションで高級感を出す
2. **空状態のイラスト** — 「大切な人を追加しましょう」のテキストだけでなく、温かみのあるイラストを追加
3. **ダークモード** — シーズンテーマに加えてダークモード対応。夜間利用時の目の負担軽減
4. **アイコンの統一感** — 絵文字ベースのアイコンを自前のSVGアイコンセットに統一すると、よりプロフェッショナルな印象に
5. **ブランドカラーの強化** — AWAIのロゴ・アクセントカラーをより一貫して使う

## 9. UI/UX面のアドバイス

1. **オンボーディング** — 初回起動時に3-4画面のチュートリアル。「大切な人を1人登録」→「記念日を入れる」→「AIに提案をもらう」の体験フロー
2. **ボトムナビゲーション** — 現在のタブは横スクロールで6つ。よく使う3-4タブをボトムナビに固定し、残りは「その他」に
3. **スワイプジェスチャー** — カード左スワイプで削除、右スワイプでピン固定
4. **入力の段階的開示** — 大切な人カードの全フィールドを最初から見せるのではなく、基本情報→詳細を段階的に
5. **フィードバックの充実** — 削除時のundo（取り消し）機能。confirm()ではなくトースト+取り消しボタン
6. **読み込み表示** — 写真読み込み時、AI提案生成時にスケルトンスクリーンやスピナー表示
