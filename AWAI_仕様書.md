## 1. プロダクト概要

### コンセプト
> 「Google広告は自分の欲を刺激する。AWAIは大切な人を喜ばせるヒントをくれる。」

AWAIは、大切な人の情報を記録・管理し、AIがギフトや体験を提案する**Personal CRMアプリ**。贈り物の履歴、相手の好み・記念日・サイズ情報などを一元管理し、最適なギフト提案をAIコンシェルジュが行う。

### ターゲットユーザー
- 贈り物を大切にする個人（家族・友人・恋人への贈答）
- 法人の贈答管理（お中元・お歳暮・取引先管理）

## 3. 機能一覧

### 3.1 タブ構成（6タブ）

| # | タブ名 | 用途 |
|---|--------|------|
| 1 | ✨ ほしいもの（wish） | 気になるアイテム管理、目的別分類（自分用/贈り物/気になる） |
| 2 | 🎀 もらった（received） | 受け取った贈り物の記録 |
| 3 | 🎁 あげた（gave） | 贈った物の記録、「ほしいもの」からのワンタップ転送 |
| 4 | 📍 行きたい場所（place） | 行きたい場所管理、カテゴリ連想・Google Maps連携 |
| 5 | 👤 大切な人（people） | 個人/法人プロフィール管理、記念日・好み・サイズ等 |
| 6 | 👥 グループ（groups） | 人物をグループ化、グループ単位でのAI提案 |

---

### 3.2 アイテム管理（ほしいもの / もらった / あげた）

**登録フィールド:**
| フィールド | 説明 |
|-----------|------|
| タイトル | アイテム名（必須） |
| 目的 | 🛒自分用 / 🎁あげたい / 💭気になる |
| 贈り先 | 誰にあげたいか（大切な人から選択） |
| URL | 商品リンク |
| 金額 | 予算・価格（¥表示） |
| タグ | 自由タグ（カンマ区切り） |
| イベント | どんな時に？（記念日名等） |
| ★評価 | 1〜5段階 |
| メモ | 自由記述 |
| 画像 | カメラ撮影 or ファイル選択（Base64保存） |
| ラベル | 色分けラベル（最大10色） |

**主要機能:**
- CRUD（追加・表示・編集・削除）
- 「🎁プレゼントする」ボタン → 相手を選択 → 日付・金額入力 → 「あげた」タブへ自動転送
- 📌ピン固定（常に上部表示）
- ⭐ランキング表示モード（評価順ソート）
- ラベルフィルタリング
- 自動ラベル付け（テキスト内容からラベル名を検出して自動適用）
- 📤共有ボタン（ほしいもの・行きたい場所のみ）

### 3.3 行きたい場所

**アイテム管理の全機能に加え：**

| 機能 | 説明 |
|------|------|
| カテゴリ連想 | 🍽食事/🎮遊び/📸観光/🛍買い物/🏨宿泊/💆リラックスの6カテゴリ |
| ジャンルタグ連動 | カテゴリ選択でジャンルタグが自動絞り込み表示（計80種以上） |
| 住所入力 | 📍住所フィールド |
| Google Maps連携 | 「🗺 Googleマップで検索」ボタン → 店名/住所でMap検索 |
| Googleマップリンク | マップアプリからURL貼り付け可 |
| 誰と行きたい？ | 大切な人・グループから複数選択 |
| ✅行った！ | 訪問済みフラグ（ワンタップ切り替え） |
| 期待度 | ★1〜5 |

**カテゴリ別ジャンルタグ例:**
- 🍽食事 → 和食、イタリアン、焼肉、寿司、ラーメン、カレー、居酒屋、バー、カフェ 他
- 🎮遊び → カラオケ、テーマパーク、映画館、キャンプ、BBQ、ゴルフ 他
- 📸観光 → 神社、お寺、温泉、絶景、美術館、水族館、世界遺産 他

### 3.4 大切な人（個人/法人）

**最も詳細なデータ管理。100以上のフィールドで人物情報を記録。**

#### 基本情報
| フィールド | 説明 |
|-----------|------|
| 種別 | 👤個人 / 🏢法人（切り替え） |
| 呼び名 | 普段の呼び方（必須） |
| 本名 | フルネーム |
| 顔写真 | アバター画像 |
| 関係 | 友人、家族、同僚等 |
| 性別 | 男/女/その他 |
| 住所 | 贈答先住所 |
| 役職 | 法人用 |
| 業種 | 法人用 |
| 関連法人 | 法人間リンク |

#### 記念日管理（複数登録可）
| フィールド | 説明 |
|-----------|------|
| 記念日名 | 例：🎂誕生日、💍結婚記念日 |
| 日付 | 年月日 / 月日のみ / 月のみ に対応 |
| 繰り返し | 毎年 / 毎月 / 単発 |
| リマインダー | ○日前に通知（複数設定可） |
| カウントダウン | 「あと○日」自動表示 |
| 経過表示 | 日数 or 年数（切り替え可） |

#### 季節の贈答（法人向け）
- お中元 🎐：毎年手配 + 予算設定
- お歳暮 🎄：毎年手配 + 予算設定

#### プロフィール詳細
| カテゴリ | フィールド |
|---------|-----------|
| 📏サイズ | 服トップス / ボトムス / 靴 / 指輪 |
| 🚬嗜好品 | タバコ（銘柄）/ お酒（種類） |
| 💖好きなもの・趣味 | 複数タグ（ピックアップ提案に使用） |
| 🎨ブランド・色 | 好みのブランド |
| 🌟推し活 | アーティスト・キャラ・チーム |
| 🍽食の好み | 好きなもの / 苦手なもの |
| ✨個性 | 性格特性メモ |
| 👨‍👩‍👧‍👦家族構成 | 名前＋メモ（複数登録） |
| 📝メモ | 自由記述 |

#### カード表示機能
- リスト表示（LINEスタイル）＋ タップで詳細展開
- 「📋 この人との記録」→ あげた/もらった/ほしいもの/行きたい場所から自動集約（最大10件）
- 各履歴アイテムをタップ → 該当タブ・カードへジャンプ（リンク機能）
- 罫線で記録エリアとアクションボタンを視覚的に分離

## 4. データ構造

### メインデータスキーマ（LocalStorage: `awai_data`）

```
{
  wish:     [ItemObject],     // ほしいもの
  received: [ItemObject],     // もらった
  gave:     [ItemObject],     // あげた
  place:    [PlaceObject],    // 行きたい場所
  people:   [PersonObject],   // 大切な人
  groups:   [GroupObject],    // グループ
  labels: {
    wish:  [{name, color}],
    gift:  [{name, color}],
    place: [{name, color}]
  }
}
```

### 各オブジェクト主要フィールド数
| オブジェクト | フィールド数 | 備考 |
|-------------|-------------|------|
| ItemObject | 15 | 全タブ共通（wish/received/gave） |
| PlaceObject | 18 | Item + 住所/地図/カテゴリ/同行者/訪問済み |
| PersonObject | 30+ | 基本情報+記念日+サイズ+嗜好+家族等 |
| GroupObject | 7 | 名前+説明+メンバーID配列 |
